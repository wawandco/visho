# Visho

Visho is a VSCode extension that provides tooling for the [Plush](https://github.com/gobuffalo/plush) templating language. 

![Preview](/assets/syntax-highlight.png)
### Features

- Syntax highlighting for `.plush.html` files
- Code snippets
